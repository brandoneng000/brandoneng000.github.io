RGB Painting Simulation

December 2016

​The RGB Painting Simulation was made to simulate the RGB color system and allow users to create pictures. The simulations come with six different canvases. Once the user applies the color to the canvas, the color cannot be removed. The cube is positioned to allow users to rotate and see their various paintings. Each face of the cube represents a different canvas. The sliders on the bottom control the RGB values and the size of the brush. The current active canvas can be outputted into a PNG file.

This program was done with C# in Unity.

It implements the RenderTexture and Texture2D from the Unity library. The RenderTextures allowed the images to be shown onto the cube. The Texture2D allowed for manipulation of the canvas.

Bottle Brigade - Mechanical Design Lead

March 2016

The Bottle Brigade was the group designated to create a baby bottle sleeve. The sleeve tracks and records the usage of the baby bottle. The encapsulation was created using AutoCAD and later 3D printed. The encapsulation houses the Raspberry Pi Zero and a battery. Its top allows wires to access the Pi Zero.

Writer's Block

January 2016

​Writer's Block was a project created during the Global Game Jam in January 2016. This was done in a three person group within a 48 hour time limit. The game was implemented with Java in Greenfoot. The player would control various Shakespeare characters against either an AI or another player. The board used procedural generation so that there would be variation with each play attempt.

Gravitatis Cecidisse

January 2016

Gravtatis Cecidisse is a text adventure puzzle game. This was implemented with Inform 7. The goal of the game is for the player to escape the house. There are various locked doors that can only be opened with a password. Each password is encoded with a cipher. The player has to solve these ciphers in order to proceed. The various ciphers used were Caesar, Atbash, A1Z26, and Vigenere.

Linear Sim

March 2015

Linear Sim is a simulation of heat transmission and spread from one or two sources across a set number of cells over a time period. This was made entirely with C. The program would be given a set of inputs that would designate the cells time frame and temperatures of one or two temperature sources. The temperature would be calculated by averaging the temperatures of adjacent cells. The designated temperatures would remain constant throughout the test.

Carrot Hunter

May 2013

Carrot Hunter is a game where the player's goal is to collect treasures hidden in the maze. This was implemented with Java in Processing. The player would collect the carrots in the maze while avoiding the rabbits that are moving around. The characters, monsters, and treasures were designed using OpenGL. The maze was created with procedural generation. The location of the monsters and treasures will vary each time the player starts a game.
